Getting started
1. Build a program with blocks

Open a category in the toolbox on the left of the workspace and drag blocks onto the canvas. Snap blocks together to control flow. Hover a block to see a tooltip explaining what it does.

2. Watch the code update

The panel on the right shows the generated MicroPython code in real time with syntax highlighting. Use Copy to send it to your clipboard or Download .py to save it.

3. Flash it onto your Pico

Install Thonny (recommended for beginners) or use mpremote, connect your RP2040/RP2350 board, paste the code or save it as main.py on the device and run.

4. Keep your work safe

Your workspace is auto-saved to your browser — reloading the page restores it. Use Export Project to produce a .xml file you can share or commit to version control, and Import Project to bring one back.

5. Learn from examples

Every example in the sidebar has an button that opens a demo view with a description, required hardware, wiring notes and expected behavior — useful even if you don't load the blocks.

Tip: The generated code uses only the standard MicroPython modules machine, time, dht and neopixel, which are included in the official MicroPython firmware for the Raspberry Pi Pico / Pico 2.
